# 批量视频上传 · 流量舱 v1.0.0

首个 Windows 桌面发行版，面向需要同时运营抖音、快手与哔哩哔哩的内容创作者和运营团队。

## 本版亮点

- Windows 10 / 11 原生桌面窗口与一键安装体验
- 批量添加视频，统一整理标题、简介、话题、分区、封面与可见范围
- 平台规格提示和智能适配任务队列
- 扫码授权交互、账号矩阵、定时发布与数据复盘界面
- 一键复制已填写内容并进入各平台官方上传页面
- 单实例运行、外链安全隔离、开始菜单及桌面快捷方式

## 安装

下载 `批量视频上传-Setup-1.0.0-x64.exe` 并运行。安装包适用于 64 位 Windows 10 和 Windows 11，默认安装到当前用户目录，不要求管理员权限。

## 使用边界

当前扫码授权、智能适配和发布队列属于桌面产品原型流程。真实自动上传需要申请各平台官方开放接口及对应资质；本软件不会绕过验证码、风控或替用户点击最终发布。

## 完整性校验

发行页同时提供 `SHA256SUMS.txt`，可用于核对下载文件未被修改。
